## 問題

$N$ 個の正整数 $a_1, a_2, \ldots, a_N$ と目標値 $T$ が与えられる。
いくつかの要素を選び、その総和を $T$ 以下でできるだけ $T$ に近づけよ。

## 入力

```
N T
a_1 a_2 ... a_N
```

## 出力

選ぶ要素数を $K$、選んだ要素の 1-indexed な番号を $i_1, i_2, \ldots, i_K$ として、以下の形式で出力せよ。

```
K
i_1 i_2 ... i_K
```

$K = 0$ の場合、2 行目は空行でも省略してもよい。

## 制約

- $1 \leq N \leq 40$
- $1 \leq T \leq 10^9$
- $1 \leq a_i \leq T$
- 同じ要素を複数回選んではならない

## スコア

選んだ要素の総和を $S$ とする。

- 出力形式が不正な場合、または $S > T$ の場合、スコアは $0$
- それ以外の場合、スコアは $\left\lfloor 10^6 \times S / T \right\rceil$

総スコアは各テストケースのスコアの合計である。

## 入力例

```
5 100
40 35 30 20 10
```

## 出力例

```
3
1 2 5
```

この出力では $40 + 35 + 10 = 85$ となる。
