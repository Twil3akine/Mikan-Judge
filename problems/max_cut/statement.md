## 問題

$N$ 頂点 $M$ 辺の重み付き無向グラフを考える。各頂点を 0 または 1 のどちらかのグループに割り当てたい。

辺 $(u, v)$ の両端が異なるグループに割り当てられているとき、その辺は「切られている」とする。
切られている辺の重み総和を最大化せよ。

ただし、グラフは入力で直接与えられるのではなく、整数 seed から以下の手順で生成される。

## グラフ生成

入力で与えられる値を `N M seed` とする。`x` を 32-bit 符号なし整数として `seed` で初期化し、以下の関数を使う。

```text
next():
    x = x xor (x << 13)
    x = x xor (x >> 17)
    x = x xor (x << 5)
    x = x & 0xffffffff
    return x
```

各辺は次のように生成する。

```text
for i = 1..M:
    a = next() % N
    b = next() % (N - 1)
    if b >= a:
        b += 1
    w = 1 + (next() % 1000)
```

このとき、頂点 `a`, `b` の間に重み `w` の辺を追加する。多重辺は存在しうる。

## 入力

```
N M seed
```

## 出力

各頂点のグループ `c_0, c_1, ..., c_{N-1}` を空白区切りで出力せよ。

```text
c_0 c_1 ... c_{N-1}
```

各 `c_i` は 0 または 1 でなければならない。

## 制約

- $2 \leq N \leq 400$
- $1 \leq M \leq 200000$
- $0 \leq seed < 2^{32}$

## スコア

切られている辺の重み総和を `cut`、全辺重み総和を `total` とする。

- 出力形式が不正な場合、スコアは $0$
- それ以外の場合、スコアは $\left\lfloor 10^6 \times cut / total \right\rceil$

総スコアは各テストケースのスコアの合計である。

## 入力例

```
6 12 1
```

## 出力例

```text
0 1 0 1 0 1
```

この出力例は一例であり、最適とは限らない。
